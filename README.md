# Мадейра 🇵🇹 — травень 2026

Інтерактивний план поїздки на Мадейру, 7–17 травня 2026.

## Що є в плані

- **План по днях** — погодинний розклад кожного дня
- **Їжа** — страви, пляжі, топ ресторани
- **Бронювання** — що і де бронювати (з пріоритетами)
- **Що взяти** — інтерактивний чекліст із збереженням стану
- **Поради** — практична інформація про острів

## Деплой на Vercel

1. Залийте репозиторій на GitHub
2. Зайдіть на [vercel.com](https://vercel.com)
3. New Project → Import Git Repository → оберіть цей репо
4. Framework Preset: **Other**
5. Deploy — готово!
